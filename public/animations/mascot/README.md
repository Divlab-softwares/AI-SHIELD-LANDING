# Mascot animated assets

Place transparent animated WebP files here using these exact names:

- `idle.webp`
- `welcome.webp`
- `think.webp`
- `point.webp`
- `celebrate.webp`
- `waiting.webp`
- `guardian.webp`
- `builder.webp`
- `ok.webp`

The app maps logical animation names to these files in `src/components/motion/mascotAssets.ts`.
