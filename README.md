# PROJET ARGUS — Landing Page

Landing page Next.js + Tailwind CSS prête pour Vercel.

## Personnalisation

Modifiez `config/site.ts` pour remplacer les liens, textes, accès de démonstration et membres de l'équipe.

Remplacez les visuels présents dans `public/screenshots/` et mettez à jour leurs chemins dans `config/site.ts` si nécessaire.

## Lancement local

```bash
npm install
npm run dev
```

Puis ouvrez http://localhost:3000.

## Vérification de production

```bash
npm run build
npm start
```
